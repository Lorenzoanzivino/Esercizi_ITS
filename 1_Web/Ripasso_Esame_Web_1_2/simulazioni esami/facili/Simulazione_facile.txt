8. (JS - Funzioni) Trasforma questa funzione classica in una Arrow Function (funzione freccia):
    function somma(a, b) {
        return a + b;
    }

    risposta:
    const somma(a, b) => a+b;

    oppure
    const somma = (a,b) => {
        return a+b;
    }

    ❌ manca "=" : const somma = (a, b) => a+b;
